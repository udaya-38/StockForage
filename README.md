## 🚀 Overview

**StockForge** is a modern Inventory Management System that helps businesses efficiently manage products, suppliers, inventory movements, and business analytics through a clean, responsive, and intuitive dashboard.

Built with modern frontend technologies, the project demonstrates scalable component architecture, state management, reusable UI, responsive design, and business-focused workflows inspired by enterprise SaaS applications.
